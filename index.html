Are Links Dying?

External link ratios across the web, 2014 → 2020 → 2024. Data: Common Crawl (306K sites) + detailed per-site analysis (170 sites, via web archive). About

About this study

Data sources: Common Crawl WAT files (306K domain-year pairs across 2014, 2020, 2024) + Wayback Machine deep-dive (170 curated domains with per-domain link analysis).

Method: For each page, all <a href> links in the raw HTML are classified as internal (same registrable domain) or external (different domain). External links are further split into social media (Twitter, Facebook, etc.) and genuine editorial links (excluding same-organisation cross-domain links like bbc.com ↔ bbc.co.uk). Domain classification uses the Public Suffix List (tldextract).

SPA detection: Pages with <300 characters of visible text in the raw HTML are flagged as potential JavaScript-rendered shells (links may exist but are invisible to crawlers).

Limitations: CC WAT captures are point-in-time snapshots; results may vary by capture date. The random sample does not cover all websites. JS-rendered pages are underrepresented in link counts.

Code & data: GitHub · paulkinlan.me
